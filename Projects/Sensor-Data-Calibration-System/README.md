# Automated Sensor Data Calibration System

## 📌 Overview

This project implements a MATLAB-based pipeline for sensor data processing, including initialization, extraction, calibration, formatting, and system modeling.

---

## ⚙️ Key Features

* Sensor metadata initialization
* Region of Interest (ROI) extraction
* Element-wise calibration
* Data padding using replication
* Data concatenation for transmission
* Linear system solution (Ax = b)

---

## 🧠 Concepts Applied

* Matrix indexing and slicing
* Element-wise operations (`.^`, `./`)
* Data replication (`repmat`)
* Concatenation (`horzcat`)
* Linear algebra (`inv`, `trace`)
